# virgin-devcontainer

A clean devcontainer setup template for development environments.

## Overview

This repository provides a minimal, clean devcontainer configuration that can be used as a starting point for new projects.

## Features

- Clean, minimal devcontainer setup
- Ready for customization
- Standard development environment structure

## Getting Started

1. Clone this repository
2. Customize the devcontainer configuration for your needs
3. Open in VS Code or GitHub Codespaces
